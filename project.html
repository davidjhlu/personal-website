Pokebank

The purpose of this app was to incentivize young people to use their savings accounts more, and in general to help them save more money.

The users would set weekly or monthly savings goals to reach, and they would be rewarded if their milestones and goals were met. User login data was stored using Firebase, and users had the option to use voice commands to deposit money.

Glasswings

The goal of this web app was to help youth with insurance pricing. Insurance felt like a subject which young people weren't really exposed to.

The web app was crowdsourced, so any users of the app could share their information, and how much they're paying for a certain kind of insurance. The model would predict prices for different kinds of insurance based on a property the user is trying to buy in a given area.

Brick Breaker

A Unity game, coded with C#, which included an interface for scoreboard, as well as a high score display.

Other functionalities included multiple levels, sound effects, gravity and also moving game objects with mouse command.

Bezier Curve Drawing Tool

A tool which mimics Adobe Illustrator, in a very basic way. Can be used to draw, colour, delete and modify bezier curves.

There was additional functionality to modify the pivot points of the Bezier Curve. Built with Java and JavaFX with Gradle being used for build automation.

Conway's Game of Life

Conway's game of life built using Java, JavaFX, and Gradle for build automation.

Pieces would be placed by the player, and the pieces would be transformed based on its neighbouring pieces. The board would periodically update at every frame. There was added functionality to create, delete and draw custom shapes, as well as start and stop the game.
